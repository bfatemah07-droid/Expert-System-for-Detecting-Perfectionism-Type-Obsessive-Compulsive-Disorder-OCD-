# Expert System for Detecting Perfectionism OCD

Simulates an expert system to detect Perfectionism-type Obsessive-Compulsive Disorder (OCD) using rule-based reasoning. Implements a questionnaire-based assessment to calculate OCD score and provide guidance.

Language: Python 3 | Execution: CLI

Features

- Rule-based reasoning for Perfectionism OCD
- Severity assessment (Mild, Strong, None)
- Provides guidance and recommendations
- Console-based interactive system

Usage
python expert_system_ocd.py
